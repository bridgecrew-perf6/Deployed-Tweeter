# Tweeter: A Twitter Clone

## Live Website
https://thirsty-heisenberg-62b60a.netlify.app/

## Functionalities
+ Authentication(SignUp and Login)
+ Email Confirmation
+ Infinite Scroll of Tweets and Comments
+ Tweet and Comment on the Tweet
+ Follow Users

+ Sendgrid(Email)
+ Django CORS
+ SQlite Database
+ Cloudinary for storing Images
